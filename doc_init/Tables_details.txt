### Table details

#### `Categories` table
| Column Name | DateType | Constraints | Description |
|---------------|------------|-------------|------ -------|
| `category_id` | INTEGER    | PRIMARY KEY | Unique category identifier |
| `name` | TEXT | NOT NULL   | Category name |
| `description` | TEXT       | Category description |

#### `ExpenseCategories` table
| Column Name   | DateType   | Constraints | Description |
|---------------|------------|-------------|------ -------|
| `expense_id`  | INTEGER | FOREIGN KEY REFERENCES `Expenses(expense_id)` | Expense ID |
| `category_id` | INTEGER | FOREIGN KEY REFERENCES `Categories(category_id)` | Category ID |

#### `SavingsGoals` table
| Column Name     | DateType    | Constraints  | Description |
|-----------------|-------------|--------------|-------------|
| `goal_id`       | INTEGER | PRIMARY KEY | Unique savings goal identifier |
| `user_id`       | INTEGER | FOREIGN KEY REFERENCES `Users(user_id)` | User ID |
| `name`          | TEXT    | NOT NULL | Name of the savings goal |
| `target_amount` | REAL    | NOT NULL | Target savings amount |
| `current_amount`| REAL    | DEFAULT 0 | Current savings amount |
| `start_date`    | DATE    | NOT NULL | Savings start date |
| `end_date`      | DATE    | Savings end date |

#### `Income` table
| Column Name    | DateType    | Constraints  | Description |
|----------------|-------------|--------------|-------------|
| `income_id`    | INTEGER     | PRIMARY KEY  | Unique income identifier |
| `user_id`      | INTEGER     | FOREIGN KEY REFERENCES `Users(user_id)` | User ID |
| `amount`       | REAL        | NOT NULL     | Amount of income |
| `source`       | TEXT        | NOT NULL     | Source of income |
| `received_date`| DATE        | NOT NULL     | Date of receipt of income |
